# dapp

> \[WIP\] Boilerplate for stack of Vue/Parcel/Electron

*Inspired by [eddyerburgh/parcel-vuejs-template](https://github.com/eddyerburgh/parcel-vuejs-template)*

- [x] Electron
- [x] Parcel
- [x] Vue
- [x] Jest
- [x] ESLint
- [x] DotEnv

### Developing

```sh
npm install
npm run dev
```

### Packaging

```sh
npm test
npm run build
```
